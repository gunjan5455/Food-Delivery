
# Food Delivery Assets

This package includes sample PNG assets for your food delivery app:
- 3 food images (pizza, burger, salad)
- Icons: home, cart, user, eye, eye-off
- 2 light-themed illustrations
- A sample React component (FoodCard.jsx)

Suggested color palette:
- Primary: #3B82F6 (Tailwind Blue-500)
- Background: #F9FAFB
- Accent: #FCD34D (Tailwind Yellow-300)
